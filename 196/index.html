196 - content: string on pseudo-element flex containers

Pseudo-element with content: string and display: flex

        .one::before {
          content: 'content value text';
          display: flex;
          width: 100%;
          height: 400px;
          justify-content: center;
          align-items: center;
          background-color: wheat;
        }

Element with content: string and display: flex

Elements can only have their content: normal, or content: content-replacement (equal to image). Note the element still reports the specified content string in getComputedStyle(element).content. But it has no effect on an element. It has the same effect as content: normal. 

      .two {
        content: 'content value text';
        display: flex;
        width: 100%;
        height: 400px;
        justify-content: center;
        align-items: center;
        background-color: wheat;
      }
