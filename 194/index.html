194 - pseudo-element flex items with display: contents

First, a pseudo-element with the default display: inline

The pseudo-element generates an inline box, and becomes a flex item of the section flex container.

Second, a pseudo-element with display: contents

The pseudo-element’s child text run replaces the pseudo-element’s box. The first child node of the section element is another text run. They are two contiguous text runs, and they form a flex item together.

Keep in mind Firefox doesn’t implement display: contents on pseudo-elements, so the pseudo-element will behave as display: inline
